| ‡∏Ç‡∏ô‡∏≤‡∏î    | Signed (‡∏°‡∏µ‡∏•‡∏ö) | Unsigned (‡∏ö‡∏ß‡∏Å‡πÄ‡∏ó‡πà‡∏≤‡∏ô‡∏±‡πâ‡∏ô) |
